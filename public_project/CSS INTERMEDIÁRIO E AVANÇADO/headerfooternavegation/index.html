Curso de Programação

JavaScript?

O JavaScript (JS) é uma linguagem de programação utilizada principalmente em páginas web. Com o JS, você pode mostrar mensagens e outras informações interessantes, fazer verificações ou mudar dinamicamente a apresentação visual das páginas, conforme o comportamento que você deseja que sua página (ou aplicação) possua.

PHP?

PHP é uma linguagem de programação gratuita e de código aberto. Apesar de ser classificada como uma linguagem de uso geral, o PHP é especialmente adequado para o desenvolvimento web e pode ser embutido em documentos HTML. O que parece uma sigla, na verdade é um acrônimo recursivo que significa Hypertext Preprocessor
